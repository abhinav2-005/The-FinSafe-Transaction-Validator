# 💳 FinSafe Transaction Validator

A simple Java console-based banking system designed to simulate secure financial transactions with proper validation and error handling.

---

## 📌 Features

- Deposit money into an account  
- Withdraw money with insufficient balance protection  
- View current account balance  
- Track complete transaction history  
- Custom exception handling  

---

## 🛠️ Tech Stack

- Language: Java  
- Concepts Used:
  - Object-Oriented Programming (OOP)
  - Exception Handling
  - Collections
  - Console Input/Output

## 🚀 Getting Started

### 🔧 Prerequisites

- Java JDK (8 or above)
- Terminal or IDE (VS Code, IntelliJ, Eclipse)

---

### ▶️ Run Locally

```bash
# Clone the repository
git clone https://github.com/abhinav2-005/The-FinSafe-Transaction-Validator

# Navigate into the project directory
cd The-FinSafe-Transaction-Validator

# Compile the program
javac Main.java

# Run the application
java Main
